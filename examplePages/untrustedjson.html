Untrusted JSON

Description

Say you have a string of valid JSON coming from an untrusted source. You'd like to be able to get a sanctified Object back without having to carry around a faceted value. All of that is fine since, assuming a decent JSON parser, nothing too shady can happen.

We'll simulate getting an untrusted value, then hand it off to JSON.parse, and then finally get back our holy object. We'll then display a specific property from the parsed object.
